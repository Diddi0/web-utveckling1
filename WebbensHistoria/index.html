Webbens Historia

När kom WWW och vem skapade det

Webbens historia startar 1989 när Tim Berners-Lee lägger grunden till

WWW, systemet som gör internet användarvänligt och

till gängligt för fler.

Teknologin bakom WWW som HTTP, URL och DNS

HTTP är en förkortning av Hyper Text Transfer Protocol som specificerar hur webbsidorna skickas.

URL är en förkortning av Uniform Resource Locator som specificerar hur webbadresser ser ut,

och DNS är en förkortning av Domain Name System som specificerar at översätter webbadresser

till IP-adresser så att datorer kan hitta och ladda rätt webbsidor.
